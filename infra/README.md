# Infra

Thư mục này ghi lại cách dựng hạ tầng lab và cấu hình platform ở mức source repo. Manifest sync thật vẫn nằm ở repo GitOps riêng.

## Subdirectories

- `k3s`: hướng dẫn dựng Kubernetes self-host/lab.
- `argocd`: ghi chú cài Argo CD và app-of-apps.
- `ingress`: Nginx Ingress, cert-manager và routing conventions.
- `observability`: Prometheus, Grafana, Loki.
- `storage`: PostgreSQL, Redis, MinIO.
- `registry`: Harbor.
