# live-service

Service quản lý live session và tích hợp MediaMTX.

## Trách Nhiệm

- Tạo live session.
- Quản lý stream key.
- Theo dõi trạng thái live.
- Tích hợp MediaMTX.

## Dependencies Dự Kiến

- PostgreSQL.
- Redis.
- MediaMTX.

## Incident Có Thể Sinh

- Livestream latency cao.
- MediaMTX quá tải.
- Sai cấu hình protocol.
- Redis/session lỗi.
