# admin-web

Next.js admin dashboard.

## Màn Hình Dự Kiến

- Users.
- Videos.
- Processing jobs.
- Live sessions.
- Incidents.
- RCA reports.
- Deployment/health overview.

## Vai Trò Trong Demo

Admin web là nơi nối product và AIOps: tạo dữ liệu sản phẩm, quan sát lỗi, mở incident và xem RCA report.
