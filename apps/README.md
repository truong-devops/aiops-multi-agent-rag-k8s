# Apps

Thư mục này chứa các ứng dụng client.

- `admin-web`: Next.js dashboard cho vận hành, quản lý video/live và xem RCA reports.
- `mobile-flutter`: mobile demo cho login, feed, upload video và xem live.

Admin web là trung tâm demo sản phẩm và AIOps. Mobile app phục vụ trải nghiệm người dùng cuối.
